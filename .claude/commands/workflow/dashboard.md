---
description: Display a status dashboard of all worktrees
---

## Workflow

1. Execute the dashboard script: `./scripts/worktree-dashboard.sh`
2. Display formatted status for each worktree

## Output

Dashboard shows for each worktree:
- Worktree name and location
- Current branch
- Number of uncommitted changes
- Number of commits ahead of main
- Clean visual formatting

## Use For

- Quick overview of all parallel work
- Identifying worktrees with uncommitted changes
- Seeing which branches need to be pushed
- Understanding overall development status
- Planning next steps
