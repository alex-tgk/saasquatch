### Writing Principles
1. **Clarity over cleverness** - Say what you mean
2. **Active voice** - "SaaSQuatch generates..."
3. **Present tense** - "The CLI creates..."
4. **Second person** - "You can configure..."
5. **Specific over generic** - Use exact tool names
